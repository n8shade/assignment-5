Question-1: What are the different ways to select an element in the DOM?

getElementById(), querySelector(), getElementsByClassName(), getElementsByTagName(), querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML: It returns the HTML content within an element.
innerText: It returns the text content of an element, along with the text content of all its descendants.
textContent: It returns the text content of an element and all its descendants.

Question-3: What is event delegation in the DOM?

Event delegation is a powerful technique in JavaScript that allows us to manage events more efficiently by taking advantage of event propagation (bubbling).

Question-4: What is event bubbling in the DOM?

Event bubbling is an important concept in the DOM (Document Object Model) related to how events propagate through the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

Creating Elements: document.createElement()
Adding Elements: appendChild(), insertBefore() or insertAdjacentElement()
Removing Elements: removeChild() or remove()
